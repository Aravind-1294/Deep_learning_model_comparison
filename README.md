# 🚀 Deep Learning Efficiency Frontier

### VGG16 vs MobileNetV2 Comparison

## 📌 Overview

This project explores the efficiency of deep learning models by
comparing VGG16 and MobileNetV2 in terms of accuracy, computation, and
real-world performance.

## 🧠 Models

-   VGG16 -- High accuracy but computationally expensive\
-   MobileNetV2 -- Lightweight and efficient

## 📊 Dataset

-   EuroSAT Dataset (Satellite image classification)

## ⚙️ Approach

-   Transfer Learning
-   Fine-tuning
-   Data Augmentation
-   Image preprocessing

## 📈 Results

-   VGG16 Accuracy: 95.93%
-   MobileNetV2 Accuracy: 97.96%

## ⚡ Key Insights

-   MobileNetV2 is \~47x more efficient
-   \~5.7x faster inference
-   Lower memory usage with higher accuracy
